## Per‑Scene Detail

> Expands each Scene Briefs row into cut-level detail. Duplicate the block below for every Scene.
> This block is the direct input source for the Seedance prompt and the storyboard sheet.

### S1 -- <TITLE>

**Model**: model/character used (referenced by char sheet UUID)
**Environment**: setting, props, art direction, lighting/mood
**Camera**: shot size (wide/medium/close) · lens feel · camera movement
**Action sequence**: step-by-step action inside the shot, in time order (0–2s … 2–5s …)
**Sound design**: physical, specific sound effects (no music — end the prompt with **No music.**)
**Ref stack**: references for this cut — storyboard sheet UUID + char sheet UUID (mandatory) + additional `--image` / `--video`
**Production Brief**: directing intent, tone, cautions
**Negative Prompt**: elements to avoid
